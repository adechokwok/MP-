# MoviePilot Telegram 通知模板合集
专为 **MoviePilot** 打造的美化版 Telegram 通知模板，包含**入库、下载、订阅、异常**全场景模板，支持丰富变量、Emoji 美化、媒体服务器同步状态展示，开箱即用。

## 模板特色
✅ 完整覆盖 **入库成功/下载完成/订阅通知/异常提醒** 四大核心场景
✅ 支持 **文件路径、分辨率、音视频编码、媒体库同步、时间戳** 等全变量
✅ Telegram 原生适配，排版整洁，带 Emoji 视觉美化
✅ 无冗余代码，可直接复制使用，支持自定义删减
✅ 兼容最新版 MoviePilot 变量体系

## 适用平台
- MoviePilot（全版本通用）
- Telegram 机器人通知
- 支持 Emby/Jellyfin/Plex 同步状态展示

## 模板清单
| 模板名称 | 适用场景 |
|---------|---------|
| 资源入库通知 | 媒体文件刮削、整理、入库完成 |
| 资源下载通知 | 种子/资源下载完成提醒 |
| 订阅更新通知 | 影视订阅更新、新剧集上线 |
| 异常错误通知 | 下载失败、入库失败、系统异常 |

## 快速使用
1. 打开 MoviePilot → 通知设置 → Telegram 通知
2. 启用 Telegram，填写 `Bot Token`、`Chat ID`
3. 复制本仓库对应模板内容，粘贴到**通知模板**输入框
4. 保存配置 → 发送测试通知 → 完成


## 常用变量说明
| 变量 | 说明 |
| :--- | :--- |
| `{{ title }}` | 影视标题 |
| `{{ year }}` | 上映/播出年份 |
| `{{ season_episode }}` | 季集信息（S01E01） |
| `{{ vote_average }}` | 评分 |
| `{{ resolution }}` | 分辨率（4K/1080p） |
| `{{ video_codec }}` | 视频编码 |
| `{{ audio_codec }}` | 音频编码 |
| `{{ file_size }}` | 文件大小 |
| `{{ file_path }}` | 文件存储路径 |
| `{{ site_name }}` | 下载来源站点 |
| `{{ media_server_name }}` | 媒体服务器名称 |
| `{{ err_msg }}` | 错误/异常信息 |

## 自定义建议
- 不需要某项信息，直接删除对应行即可
- 可修改 Emoji、文字描述、时间格式
- 可新增/替换变量，参考 MoviePilot 官方变量文档

## 问题排查
- 收不到通知：检查 Token、Chat ID、网络代理
- 变量为空：当前任务无对应数据，删除该行即可
- 格式错乱：确保 JSON 格式无语法错误（引号、逗号完整）
